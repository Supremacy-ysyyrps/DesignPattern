### 文件介绍
Builder      : 构造者模式
config       : 配置文件, 设置图标族
FJEFactory   : 抽象工厂类和具体工厂类
FJEProduct   : 抽象产品类和具体产品类
JSONComponent: 组合模式
main         : 客户端
SuperFactory : 抽象工厂类
### 执行方式：
```bash
python main.py -f test.json -s [tree/rect] -i [default/fruit/animal/sports]
```
e.g.
```bash
python main.py -f test.json -s tree -i fruit
```
